Password Checker

This Python script uses the HIBP API to check if passwords have been exposed in data breaches, applying K-anonymity for user privacy. It securely hashes passwords with SHA-1 and sends only the first five characters of the hash to the API. It retrieves potential matches and checks if the full hash exists. Importantly, passwords aren’t saved and are deleted once you close the GUI as the code runs on your computer, ensuring maximum security.

Skills used

Python, HIBP API, Hashing, K-Anonymity and TKinter GUI
